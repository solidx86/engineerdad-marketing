Phase 1 — Data model & two-layer foundation

  • 1.1 Schema. Add claim_bindings jsonb (default [], not null) to scripts in schema.ts. pnpm db:sandbox → pnpm db:generate; commit schema.ts + generated packages/*/drizzle/ together (pnpm lint:migrations).
  • 1.2 Zod. ClaimBindingSchema in zod.ts (claim, kind enum, chartRef nullable, figures[], takeaway, gapNote nullable) with refinements: data ⇒ chartRef set; gap ⇒ gapNote set, chartRef null; qualitative ⇒ chartRef null. Wire onto Script shape. Unit tests per refinement.
  • 1.3 Numeric-normalize util. New packages/shared/src/numeric/normalize.ts — parse RM/k/M/%/age/plain → canonical number; tracesTo(figure, haystack, tol). Table-driven tests (RM1.2M→1_200_000, "age 60"→60, "41%"→0.41, rounding tolerance).
  • 1.4 Chart-metadata loader. Reads a chart YAML → {id,title,scenario,labels,valueRanges,source_citation}. Tests over real YAMLs.
  • 1.5 corpus.list_charts MCP. Add to mcp-servers/corpus + packages/corpus/src/tools.ts — live readdir of corpus/data/charts/ through the loader. NOT index-backed. Tests.
  • 1.6 datasets/ rename. git mv corpus/data/*.json corpus/data/datasets/. Update every consumer from 0.1. Add the provenance contract (new charts cite a dataset path; existing grandfathered). Re-run touched tests.
Gate: pnpm -r build clean; new unit suites green; db:sandbox applied.
