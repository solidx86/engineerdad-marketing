  1. New packages/<name>/ dir with package.json + tsconfig.json + src/index.ts.
  2. Move business-logic files from mcp-servers/<name>/src/ to packages/<name>/src/.
  3. Re-export tool-shaped functions from packages/<name>/src/index.ts.
  4. Rewrite mcp-servers/<name>/src/index.ts as ~30-line stdio adapter (pattern = mcp-servers/store/src/index.ts).
  5. Add deps entry, dispatch rows, update ExecDeps.
